# 🕵️ TruthLens — Fake News Detector

> A modern fake news detection web application that analyzes headlines and article content using credibility heuristics, linguistic patterns, and misinformation indicators.

🌐 Live Demo: https://truthlens.netlify.app 

---

## 🚀 Features

- Analyze news headlines and article text
- Detect clickbait and sensational language
- Identify conspiracy-related phrases
- Credibility scoring system
- Animated visual score meter
- Verification guidance for users
- Trusted fact-checking source links
- Responsive futuristic UI
- Fully client-side application

## ⚙️ How It Works

TruthLens uses a rule-based credibility analysis system to evaluate content.

The application checks for:
- Sensational headlines
- Emotional trigger phrases
- Suspicious wording patterns
- Overconfident claims
- Credibility indicators
- Balanced reporting signals

Based on these signals, the system generates:
- Credibility score
- Risk verdict
- Verification suggestions

> Note:  
> This project currently uses a heuristic/rule-based analysis approach.  
> AI/ML-based NLP models may be integrated in future versions.
